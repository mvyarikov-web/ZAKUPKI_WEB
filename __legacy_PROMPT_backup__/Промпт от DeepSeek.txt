ПРОМПТ ДЛЯ АНАЛИЗА ТЕНДЕРНОЙ ДОКУМЕНТАЦИИ

**ЗАДАЧА:** Проанализируй предоставленный текст тендерной документации и дай структурированный ответ ТОЛЬКО на основе реальных данных.

**СТРОГИЕ ТРЕБОВАНИЯ:**
- Запрещено придумывать несуществующее оборудование, ссылки или контакты
- Если информация отсутствует в документе - указывать "Не найдено в документации"
- Все утверждения должны подтверждаться цитатами из текста

---

**РАЗДЕЛ 1: ОСНОВНАЯ ИНФОРМАЦИЯ**
- Краткая выжимка о тендере (предмет, сроки, бюджет)
- Контакты заказчика (извлекать ТОЛЬКО из текста документа)
- Требования к поставке и срокам

**РАЗДЕЛ 2: ОБОРУДОВАНИЕ И ХАРАКТЕРИСТИКИ**
Перечисли ВСЕ позиции оборудования с точными характеристиками, указанными в документации. Формат:
- Наименование: [точное название из документа]
- Характеристики: [параметры из документа]
- Количество: [указанное количество]
- Требования: [особые требования]

**РАЗДЕЛ 3: ТРЕБОВАНИЯ К МОНТАЖУ**
- Указаны ли требования к монтажу: Да/Нет/Частично
- Если да - перечисли конкретные требования
- Цитаты из документа, подтверждающие вывод

**РАЗДЕЛ 4: АНАЛИЗ НА РИСКИ**
Проанализируй документ на предмет потенциальных рисков:
- Непрозрачные формулировки в технических требованиях
- Завышенные/заниженные сроки
- Нестандартные условия оплаты
- Противоречия в документации
- ОЦЕНКА: Низкий/Средний/Высокий риск (с обоснованием)

---

**ВАЖНО ДЛЯ ПОДБОРА ОБОРУДОВАНИЯ:**
Если требуется подбор моделей - использовать ТОЛЬКО реальные данные:
- Поиск осуществлять через доступные базы данных
- Ссылки предоставлять ТОЛЬКО при 100% совпадении характеристик
- Если точных аналогов нет - указать "Точные аналоги не найдены"
- Для предложения аналогов указывать РЕАЛЬНЫЕ расхождения в характеристиках

**ФОРМАТ ОТВЕТА:**
- Четкая структура по разделам
- Цитаты из документа в скобках []
- Отметки "Не указано" для отсутствующей информации
- Честное "Информация не найдена" вместо вымысла

Начни анализ.
