## Миграция промптов (кратко)
1. Создать таблицу `prompts` (id, owner_id, name, text, tags JSONB, created_at, updated_at).
2. Написать скрипт `scripts/migrate_prompts.py`, который:
	- читает файлы из `PROMPT/` (включая подпапки),
	- сохраняет содержимое как строки в БД, `name` = относительный путь без расширения,
	- добавляет простые теги по имени папки,
	- пропускает пустые/битые файлы с логированием.
3. Обновить UI/бэкенд, чтобы при обращении к промптам использовать БД; для обратной совместимости оставить только чтение (без записи) из файлов, если БД недоступна.
4. После успешной миграции — архивировать содержимое `PROMPT/` или оставить `.gitkeep` и очистить файлы.
5. Добавить минимальные тесты: импорт одного файла, дубликаты имён, теги по пути.
