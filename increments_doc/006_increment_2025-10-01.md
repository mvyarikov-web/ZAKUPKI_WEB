# Постановка

Добавить специальные маркеры начала и конца документа в индекс для упрощения и ускорения поиска содержимого при превью. Обеспечить работу с архивами только на этапе индексации, все дальнейшие операции (поиск, превью) — исключительно по файлу индекса.

# Спецификация
- В индекс `_search_index.txt` добавлены маркеры `<<< НАЧАЛО ДОКУМЕНТА >>>` и `<<< КОНЕЦ ДОКУМЕНТА >>>` для каждого документа.
- Функция извлечения текста для превью теперь ищет содержимое только между этими маркерами.
- Архивы (ZIP/RAR) обрабатываются только при построении индекса, дальнейшая работа — только с индексом.
- Старый формат индекса поддерживается для обратной совместимости.
- Вся логика поиска и превью теперь не обращается к исходным файлам/архивам, только к индексу.

# Реализация (простыми словами)
- В модуле `indexer.py` добавлены маркеры в функцию записи индекса.
- В роуте превью (`pages.py`) обновлён парсер: поиск текста между маркерами, fallback на старый формат.
- Пересоздан индекс, протестировано на архивных и обычных файлах.
- Все изменения закоммичены и отправлены в репозиторий.
